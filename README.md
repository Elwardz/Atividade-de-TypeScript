## 📌 Descrição

Este projeto contém a implementação dos **21 exercícios propostos na Atividade 01** da disciplina de Programação para Web, utilizando **TypeScript**. Os exercícios abordam funções básicas, manipulação de arrays e objetos, uso de tipos, estrutura condicional, laços de repetição e muito mais.

---

## 🚀 Como Executar

### ✅ Pré-requisitos

- **Node.js** instalado: [https://nodejs.org](https://nodejs.org)
- **TypeScript** instalado globalmente:

```bash
npm install -g typescript
